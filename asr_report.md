# 选型理由（最终选择Vosk中文小模型）
1. 部署难度最低：仅需1行pip安装，模型自动下载，无需手动配置路径、环境变量，新手开箱即用，助教可快速复现；
2. 轻量高效：模型体积仅500MB，纯CPU推理极快，笔记本无性能压力，适配作业的PC实测要求；
3. 功能全覆盖：原生支持流式/实时识别，可同时完成「本地音频识别」和「麦克风实时识别」两种实验方式，比Whisper（需二次开发实现实时）、Sherpa-ONNX（功能单一）更适配作业要求；
4. 中文适配好：全覆盖普通话识别，准确率高，适合本次实验的中文配音文稿识别场景；
5. 协议友好：Apache 2.0开源免费，无商业使用限制，符合课程实验的开源要求。
